# MoyuDynamicTp
一个MC插件，实现泰坦陨落2游戏中，时空穿梭关卡的无缝传送效果，在 对玩家进行传送到 除了部分细节外，两张一模一样的另一区域时，玩家，基本不会发觉位置、视角的变动

# 使用说明
您需要自行下载龙核最新版本，并将其放在项目根目录的libs文件夹下，libs文件夹需要自己创建，并更改idea中的依赖为新下载的龙核路径

# 进阶使用办法
你可以借助NI物品插件或龙核按键指令，把此插件指令绑定在物品或按键上，右键物品或按下按键即可切换
