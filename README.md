# MoyuDynamicTp
一个MC插件，实现泰坦陨落2游戏中，时空穿梭关卡的无缝传送效果，在除了部分细节外，两张一模一样的区域 对玩家进行传送到另一张地图时，玩家，基本不会发觉位置、视角的变动

# 使用说明
您需要自行下载龙核最新版本，并将其放在项目根目录的libs文件夹下，需要自己创建libs文件夹，并更改idea中的依赖为新下载的龙核路径
